| 類別         | 用法            | 範例       |
| ------------ | --------------- | ---------- |
| **主文字**   | `text-text`     | 深灰或白   |
| **副文字**   | `text-muted`    | 淺灰       |
| **強調文字** | `text-primary`  | 主題色     |
| **錯誤文字** | `text-danger`   | 紅色       |
| **成功提示** | `text-success`  | 綠色       |
| **禁用文字** | `text-disabled` | 灰階透明度 |

✅ 最常見的配色範例（視覺搭配）
| 元素 | 背景 | 文字 | 補充 |
| -------- | ------------------------------- | ---------------------- | ------------ |
| 主按鈕 | `bg-primary` | `text-text-on-primary` | 文字白 |
| 次按鈕 | `bg-transparent border-primary` | `text-primary` | hover 時反轉 |
| 卡片 | `bg-surface` | `text-text` | 陰影輕微 |
| 警告訊息 | `bg-warning/20 border-warning` | `text-warning` | 提示感 |
| 錯誤訊息 | `bg-danger/20 border-danger` | `text-danger` | 顯眼警示 |
| 背景 | `bg-background` | — | |

✅ 總結
| 分類 | 顏色變數 | 功能 |
| -------- | --------------------------------------------- | ------------------ |
| 品牌主色 | `--primary` | 主操作、主強調 |
| 次要色 | `--secondary` | 次要區塊或邊框 |
| 狀態色 | `--success`, `--warning`, `--danger` | 表單回饋、狀態提示 |
| 背景色 | `--background`, `--surface` | 整體與卡片底色 |
| 文字色 | `--text`, `--text-muted`, `--text-on-primary` | 不同層級文字顏色 |
