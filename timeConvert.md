| 想要的时长                  | Max-Age 设置        |
| --------------------------- | ------------------- |
| 30 秒                       | `30`                |
| 1 分钟                      | `60`                |
| 5 分钟                      | `60 * 5`            |
| 1 小时                      | `60 * 60`           |
| 8 小时（企业后台常用）      | `60 * 60 * 8`       |
| 1 天                        | `60 * 60 * 24`      |
| 7 天（常见 Remember Login） | `60 * 60 * 24 * 7`  |
| 30 天（常见自动登录）       | `60 * 60 * 24 * 30` |
